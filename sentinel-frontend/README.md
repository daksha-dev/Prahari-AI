# Sentinel Dashboard

AI Security Analyst for IoT Networks.

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Recharts
- Framer Motion
- Lucide React

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Environment Variables
- `VITE_API_URL`: Set to your backend API URL. If unset or "mock", the app uses simulated data.

## Features
- **Real-Time Overview**: Monitor network trust and device fleet status.
- **Deep Device Insights**: Analyze trust timelines, drift signals, and behavioral heatmaps.
- **AI Analyst**: Interactive chat to explain anomalies and recommend actions.
- **Evidence Cards**: Granular technical evidence for every detected incident.
